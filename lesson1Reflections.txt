Using diff to Find Bugs
Mit den Befehlen diff unter Linux, Mac oder cp unter Windowssystemen können zwei Files verglichen werden. 
Dabei werden unterschiedliche Stellen angezeigt. So sind Fehler oder Änderungen im Code leicht auffindbar.

Using Git to View History:
How can you use the commands git log and git diff to view the history of files?
==============================
Mit git log erhält man eine Liste von Commits. Anhand der Beschreibung sieht man wer, was, wann geändert hat. Vergleicht man mit git diff id_old id_new zwei Commits, werden die Unterschiede dieser zwei Versionen angezeigt, zB ob Code Zeilen hinzu oder entfernt wurden.
